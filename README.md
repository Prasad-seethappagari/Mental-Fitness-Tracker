# 🧠 Mental Fitness Tracker

A Machine Learning–based Mental Fitness Tracker application designed to analyze mental health data and predict mental fitness levels using data science and regression models.

This project helps users understand mental wellness patterns by analyzing various mental health factors and generating predictive insights through AI and Machine Learning techniques.

---

# 🚀 Features

- 📊 Mental fitness prediction using Machine Learning
- 🧠 Data analysis and visualization
- 📈 Regression model implementation
- 🔍 Exploratory Data Analysis (EDA)
- 🧹 Data preprocessing and cleaning
- 📉 Graphical insights and trend analysis
- 🤖 AI-powered prediction system
- 📋 User-friendly workflow for mental health analysis

# 📊 Problem Statement

Mental health is an important part of overall well-being. This project aims to analyze mental health-related datasets and predict mental fitness scores using machine learning algorithms.

The system processes user-related mental health data and generates predictive insights that can help in understanding mental wellness trends.

# ⚙️ Installation & Setup

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/Prasad-seethappagari/Mental-Fitness-Tracker.git

cd Mental-Fitness-Tracker
```

---

# 🔹 Create Virtual Environment (Optional)

## Windows

```bash
python -m venv venv
venv\Scripts\activate
```

## Mac/Linux

```bash
python3 -m venv venv
source venv/bin/activate
```

---

# 🔹 Install Dependencies

```bash
pip install -r requirements.txt
```

If `requirements.txt` is unavailable:

```bash
pip install pandas numpy matplotlib seaborn scikit-learn jupyter
```

---

# ▶️ Run the Project

## Start Jupyter Notebook

```bash
jupyter notebook
```

Open:

```text
Mental_Fitness_Tracker.ipynb
```

Run all notebook cells step by step.
